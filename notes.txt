CSS - Cascading style sheet
----------------------------


3 ways to define CSS

1. inline  - style="background-color: dodgerblue; text-align: center; color: white"
  >> style attribute inside html tags

2.internal styling = <style></style>   => tag in <head> tag
     <style>
        p {
            background-color: pink;
        }
    </style>

3. external styling = <link> tag calling css file
    <link rel="stylesheet" href="style.css" />
